# Props 参数

| 参数              | 类型     | 默认值            | 必填 | 说明                 |
|-------------------|----------|-------------------|------|----------------------|
| `currentPage`     | `number` | —                 | 是   | 当前页码              |
| `totalPages`      | `number` | —                 | 是   | 总页数                |
| `maxVisiblePages` | `number` | `5`               | 否   | 最多显示的页码数量      |

---

# Events 事件

| 事件名  | 回调参数      | 说明             |
|---------|-------------|------------------|
| `change`| `page: number` | 页码变化时触发，返回新的页码 |

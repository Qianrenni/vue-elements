# Props 参数

| 参数         | 类型        | 默认值    | 必填 | 说明                |
|------------|-----------|--------|----|-------------------|
| `content`  | `string`  | `''`   | 是  | 需要渲染的 Markdown 内容 |
| `showCopy` | `boolean` | `true` | 否  | 是否显示复制按钮          |

| `getContent` | —            | `string`                                             | 获取原始传入的 Markdown 内容 |

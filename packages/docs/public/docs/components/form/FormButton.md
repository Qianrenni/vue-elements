# Props 参数

| 参数        | 类型                              | 默认值     | 必填 | 说明         |
| ----------- | --------------------------------- | ---------- | ---- | ------------ |
| `type`      | `'button' \| 'submit' \| 'reset'` | `'button'` | 否   | 按钮类型     |
| `disabled`  | `boolean`                         | `false`    | 否   | 是否禁用     |
| `autofocus` | `boolean`                         | `false`    | 否   | 是否自动聚焦 |
| `size`      | `'large' \| 'middle' \| 'small'`  | `'middle'` | 否   | 按钮大小     |

---

# Events 事件

| 事件名  | 回调参数   | 说明       |
| ------- | ---------- | ---------- |
| `click` | `e: Event` | 点击时触发 |

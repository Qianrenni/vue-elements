# Props 参数

| 参数         | 类型                 | 默认值  | 必填 | 说明           |
| ------------ | -------------------- | ------- | ---- | -------------- |
| `modelValue` | `string \| string[]` | `[]`    | 否   | 激活的面板     |
| `accordion`  | `boolean`            | `false` | 否   | 是否手风琴模式 |

---

# Events 事件

| 事件名              | 回调参数                    | 说明         |
| ------------------- | --------------------------- | ------------ |
| `update:modelValue` | `value: string \| string[]` | 值变化时触发 |

---

# Slots 插槽

| 插槽名    | 说明       | 示例                                                     |
| --------- | ---------- | -------------------------------------------------------- |
| `default` | 折叠面板项 | `<Collapse><CollapseItem>内容</CollapseItem></Collapse>` |

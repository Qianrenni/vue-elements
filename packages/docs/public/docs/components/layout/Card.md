# Slots 插槽

| 插槽名    | 说明     | 示例                                    |
| --------- | -------- | --------------------------------------- |
| `header`  | 卡片头部 | `<template #header>头部内容</template>` |
| `left`    | 卡片左侧 | `<template #left>左侧内容</template>`   |
| `default` | 卡片主体 | `<Card>主体内容</Card>`                 |
| `right`   | 卡片右侧 | `<template #right>右侧内容</template>`  |
| `footer`  | 卡片底部 | `<template #footer>底部内容</template>` |

# Props 参数

| 参数     | 类型               | 默认值   | 必填 | 说明     |
| -------- | ------------------ | -------- | ---- | -------- |
| `src`    | `string`           | —        | 是   | 图片地址 |
| `alt`    | `string`           | `''`     | 否   | 替代文本 |
| `width`  | `string \| number` | `'100%'` | 否   | 图片宽度 |
| `height` | `string \| number` | `'100%'` | 否   | 图片高度 |

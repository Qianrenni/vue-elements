# Props 参数

| 参数    | 类型                                                                  | 默认值   | 必填 | 说明                         |
| ------- | --------------------------------------------------------------------- | -------- | ---- | ---------------------------- |
| `type`  | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'gray' \| 'info'` | `'info'` | 否   | 徽章类型，决定颜色样式       |
| `value` | `number \| string`                                                    | `1`      | 否   | 徽章显示的值                 |
| `max`   | `number`                                                              | `99`     | 否   | 最大值，超过此值显示为 `99+` |
| `dot`   | `boolean`                                                             | `false`  | 否   | 是否显示为小圆点模式         |

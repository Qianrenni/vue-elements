# Props 参数

| 参数         | 类型                                                          | 默认值         | 必填 | 说明        |
|------------|-------------------------------------------------------------|-------------|----|-----------|
| `type`     | `'primary' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'gray'` | `'info'`    | 否  | 标签类型，影响颜色 |
| `value`    | `number \| string`                                          | `1`         | 否  | 显示的值，支持数字和字符串 |
| `max`      | `number`                                                    | `99`        | 否  | 最大值，超过则显示为 `99+` |
| `dot`      | `boolean`                                                   | `false`     | 否  | 是否显示为小圆点 |

> 💡 提示：支持原生 `span` 的所有属性（如 `class`, `style` 等）

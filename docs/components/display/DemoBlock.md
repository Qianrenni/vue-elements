# Props 参数

| 参数   | 类型     | 默认值 | 必填 | 说明       |
|--------|----------|--------|------|------------|
| `title`| `string` | `''`   | 否   | 示例标题   |
| `code` | `string` | `''`   | 否   | 可复制代码内容 |

# Props 参数

| 参数     | 类型   | 默认值 | 必填 | 说明         |
|--------|------|------|----|------------|
| `title` | string | undefined | 否  | 示例标题，用于显示在 Demo 块头部 |
| `code`  | string | ``    | 否  | 要展示和复制的代码内容 |

# Props 参数

| 参数      | 类型                                              | 默认值         | 必填 | 说明                      |
|---------|-------------------------------------------------|-------------|----|-------------------------|
| `percent`  | `string`                                        | `undefined` | 是  | 进度条百分比，例如：'50%'       |
| `color`    | `string`                                        | `'var(--primary-color)'` | 否  | 进度条颜色，支持 CSS 变量或颜色值 |
| `direction`| `'horizontal' \| 'vertical'`                  | `'horizontal'` | 否  | 进度条方向，水平或垂直           |

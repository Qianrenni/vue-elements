# Props 参数

| 参数             | 类型        | 默认值     | 必填 | 说明               |
|----------------|-----------|---------|----|------------------|
| `scollX`       | `boolean` | `false` | 否  | 是否启用横向滚动         |
| `scollY`       | `boolean` | `false` | 否  | 是否启用纵向滚动         |
| `threshold`    | `number`  | `20`    | 否  | 触发 ended 事件的阈值距离 |
| `emitInterval` | `number`  | `300`   | 否  | 事件触发间隔时间（毫秒）     |

---

# Events 事件

| 事件名     | 回调参数 | 说明                            |
|---------|------|-------------------------------|
| `ended` | —    | 当滚动接近容器末尾时触发（根据 threshold 设置） |

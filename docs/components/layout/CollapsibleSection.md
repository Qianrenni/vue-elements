# Props 参数

| 参数             | 类型                                                          | 默认值         | 必填 | 说明                     |
|------------------|-------------------------------------------------------------|-------------|----|--------------------------|
| `isShowArrow`    | `boolean`                                                   | `true`      | 否  | 是否显示箭头按钮            |
| `initialExpanded`| `boolean`                                                   | `true`      | 否  | 初始展开状态               |
| `direction`      | `'up' \| 'down' \| 'left' \| 'right'`                       | `'down'`    | 否  | 展开方向，影响箭头位置和动画 |

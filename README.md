# 快速开始

**1.引入组件库**

```
npm install qyani-components@latest
```

**2.引入样式**

```
import 'qyani-components/dist/style.css';
```

**3.引入组件**

```
import { the-components } from 'qyani-components';
```

**4.全局安装**

```
import qyanicomponents from 'qyani-components
app.use(qyanicomponents)
```

# 打包

```shell
pnpm run build
```

# 测试

```shell
pnpm run dev
```

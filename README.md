# 快速开始

**1.引入组件库**

```
npm install qyani-components@latest
```

**2.引入样式(任意.css结尾)**

```
import 'qyani-components/dist/qyani-components.*.css';
```

**3.引入组件**

```
import { the-components } from 'qyani-components';
```

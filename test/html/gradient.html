🎨 CSS 渐变完整用法演示

线性、径向、锥形、重复渐变 + 调色技巧 + 动画

1. 线性渐变（Linear Gradient）

to right: red → blue
background: linear-gradient(to right, red, blue);
45deg: yellow → green → blue
background: linear-gradient(45deg, yellow, green, blue);
彩虹色谱（指定位置）
background: linear-gradient(to bottom, red 0%, orange 20%, yellow 40%, green 60%, blue 80%, indigo 90%, violet 100%);
渐变叠加图片（遮罩效果）
background: linear-gradient(transparent, rgba(0,0,0,0.8)), url('...');
background-size: cover;

2. 径向渐变（Radial Gradient）

圆形：红粉渐变
background: radial-gradient(circle, #ff6b6b, #c4456d);
椭圆：左上角开始
background: radial-gradient(ellipse at top left, #a8edea, #fed6e3);
大小控制：closest-side
background: radial-gradient(closest-side, #fff, #000);

3. 锥形渐变（Conic Gradient）

彩色环（色相过渡）
background: conic-gradient(red, yellow, lime, teal, blue, magenta, red);
HSL 色相循环
background: conic-gradient(from 0deg at 50% 50%, hsl(0,100%,50%), hsl(60,100%,50%), ..., hsl(300,100%,50%));

4. 重复渐变（Repeating Gradients）

45° 条纹
repeating-linear-gradient(45deg, #60a5fa, #60a5fa 10px, #3b82f6 10px, #3b82f6 20px)
同心圆点阵
repeating-radial-gradient(circle at 20px 20px, #ff6b6b 0px, #ff6b6b 5px, #ff8e8e 5px, #ff8e8e 10px)

5. 渐变动效

流动渐变背景
@keyframes slideGradient {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

background-size: 400% 100%;
animation: slideGradient 8s ease infinite;

✅ 支持现代浏览器（Chrome、Edge、Firefox、Safari）
⚠️ 锥形渐变在旧版浏览器中可能不支持
